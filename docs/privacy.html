Privacy Policy for Zoomies

Effective date: April 1, 2026

Zoomies is designed to monitor AI output status locally in the browser and does not collect, store, or transmit chat content or personal data.

Zoomies ("the Extension") is a Chrome browser extension that monitors whether supported AI websites are actively generating output and displays that status visually to the user through a small on-page cat indicator, floating panel, and extension badge.

Your privacy is important. This Privacy Policy explains what data Zoomies does and does not collect, how it works, and how user information is handled.

1. What Zoomies Does

Zoomies detects whether supported AI websites, such as ChatGPT, Gemini, Claude, and DeepSeek, are currently generating output. It presents this status locally in the browser using:

  • a floating cat indicator
  • a collapsible panel
  • an extension badge

The Extension is designed only for AI output status monitoring and does not provide analytics, advertising, profiling, or content collection features.

2. Information Zoomies Collects

Zoomies does not collect, transmit, sell, or share personal data, chat content, prompts, responses, or other user-generated text.

Zoomies only stores a small amount of local extension settings in the browser, such as:

  • which supported sites are enabled or disabled
  • whether debug mode is enabled
  • whether the cat panel is collapsed

These settings are stored locally using Chrome extension storage and are used only to provide the Extension's core functionality and user preferences.

3. Information Zoomies Does Not Collect

Zoomies does not collect or process the following for transmission or sale:

  • personally identifiable information
  • health information
  • financial or payment information
  • authentication credentials
  • personal communications
  • location data
  • browsing history for advertising or profiling purposes
  • chat prompts or AI responses
  • page text, images, or other website content for remote storage

The Extension does not read or store user conversations as a product feature.

4. How Zoomies Works on Web Pages

Zoomies uses content scripts on supported websites to determine whether an AI site is currently generating output. This detection is based on page state signals such as interface controls and activity patterns.

This processing happens locally in the user's browser and is used only to update the visual status shown by the Extension.

Zoomies does not send page content or chat data to any external server.

5. Data Sharing

Zoomies does not sell, rent, transfer, or share user data with third parties.

No user chat content, prompts, responses, or personal information is transmitted to the developer or to outside services.

6. Remote Code

Zoomies does not use remotely hosted code for its core functionality. Its executable code is packaged with the Extension.

7. Data Security

Because Zoomies does not collect or transmit personal data to external servers, privacy risk is minimized by design. Any locally stored preference data remains in the user's browser environment.

8. Children's Privacy

Zoomies is not directed to children under 13 and is not designed to collect personal information from children.

9. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time to reflect changes to the Extension or applicable requirements. If changes are made, the updated version will be posted at the same URL with a revised effective date.
